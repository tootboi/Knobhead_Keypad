# Knobhead Keypad
A 3x3 keypad with a rotary encoder.

### What can it do?
Media keys (volume up, down, playe/pause), macros (ctrl + Z, ctrl + shift + esc) or ordinary keyboard keys.

### Hardware needed
* 1x Pro Micro
* 1x EC11 rotary encoder
* 9x mechanical switches
* 9x keycaps
